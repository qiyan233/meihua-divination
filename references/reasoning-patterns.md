# Reasoning Patterns

用这份规则把五个象位稳定地转成可解释的中文断事。

## 核心映射

- **主象**：相当于体，代表问事者自身状态、根基、主导心气
- **客象**：相当于用，代表眼前之事、外部条件、对手或对方
- **阻象**：不是传统原词，可视作“过程卡点 / 中程阻滞”的现代化支架
- **变象**：看后势、终势、事情后续往哪边转
- **应象**：看更合适的应法、解法、落手点

## 推演顺序

### Step 1：先看主象
问：
- 这件事的根气是什么？
- 是开路、生发、加速、承压、迟滞，还是回收蓄势？

### Step 2：再看客象
问：
- 外部到底是在扶主，还是在压主？
- 这是机会先来，还是压力先到？
- 这股外势是眼前就显，还是虽有其象但尚未坐实？

### Step 3：再看阻象
问：
- 真正的卡点是什么？
- 是现实承接、边界规则、情绪耗损、节奏不对，还是时机未到？
- 这是硬障，还是看着吓人但力量其实不重？

### Step 4：再看变象
问：
- 如果照现在这样发展，后势会转松、转紧、转散、转稳，还是转硬？
- 是越往后越好，还是前面尚可、后面反而吃力？

### Step 5：最后看应象
问：
- 最合适的应法是什么？
- 该推进、观望、试探、收口、沟通、止损、立边界，还是先补盘？

## 近势 / 中势 / 后势

吸收传统里“用为近应，互为中程，变为后势”的意思，但做简化映射：

- **近势**：优先看客象，判断事情眼前怎么显
- **中势**：优先看阻象，判断过程里卡在哪里
- **后势**：优先看变象，判断事情后面往哪边转

因此写 `断势` 时，优先使用：
- 近势：……
- 中势：……
- 后势：……

不必每次都写得很长，但最好能体现阶段感。

## 生克判断：不要机械背表

### 基本解释

- **客生主**：外缘能扶主，事情对问事者有托举
- **客克主**：外部条件对问事者形成压制
- **主生客**：问事者持续往事里耗力，容易主气外泄
- **主克客**：不是不能处理，但多靠自己出手，成本偏高
- **比和**：同气相引，会放大当前倾向

### 轻重判断

不要把“生克”直接等于绝对吉凶。至少要同时看：

- **强 / 弱**：强则作用更实，弱则作用较轻
- **动 / 静**：动则更快显化，静则尚在潜伏或拖延
- **顺 / 逆**：顺则容易成势，逆则即便有象也要折损一层

可记成一句：

> 生克给方向，强弱给分量，动静给显化快慢。

## 动静规则

### 不动不占，不因事不占

对 skill 的实际用法是：
- 尽量一问一事
- 纯好奇、无现实牵引的问题，输出宜轻
- 若问题本身没有明确事端，不要硬装成大凶大吉

### 动静怎么用

- **动**：表示事情已在显化、逼近、推动、加速
- **静**：表示事情还在积蓄、停顿、潜伏、观望

这里主要用于：
- 判断事情是偏快、偏慢、还是渐显
- 判断是宜先动、宜先守、还是宜轻试

**不要**把它包装成精确应期机器。
这个 skill 更适合说：
- 应得较快
- 近期会有回音
- 中势才见分晓
- 后势仍需等待

## 常见结论模板

### 何时说“宜动”
满足其中多数：
- 主象有生发或推进性
- 客象扶主或至少不强压
- 阻象不是重压型
- 变象显示后势能落地
- 应象支持试跑、沟通、推进

### 何时说“宜守”
满足其中多数：
- 主象偏土、水或明显收束
- 客象与阻象都在压主或耗主
- 变象未见舒展，反而提示继续收口
- 应象更适合观望、蓄势、先补盘

### 何时说“可动但不宜急”
满足其中多数：
- 主象本身并非全静
- 但阻象显示承接不足、节奏不稳、边界混乱或内耗过重
- 变象说明后势要靠调整而非猛冲
- 应象适合分段行动、轻试、先验证

### 何时说“外顺内阻”
满足其中多数：
- 客象看起来对你不坏
- 但阻象显示内部结构未稳
- 常见于：项目表面可行、关系表面有回响、运势表面有路，但真正卡在承接盘

### 何时说“先破后立”
满足其中多数：
- 金或火对旧的土/水滞局形成切割
- 短期摩擦不可避免
- 但变象与应象都提示：只要敢清旧耗，后势会比现在明

## 模式侧重点

### Career / project
重点看：
- 盘面是否坐稳
- 是否能先试跑再放大
- 外机来了能不能接住
- 承接成本是否过高

### Relationship
重点看：
- 节奏是否对齐
- 是真冷却，还是气口收紧
- 主动沟通会修复还是加压
- 应法应偏轻、偏柔还是偏断

### Fortune / phase
重点看：
- 卡点在何处
- 哪里在漏气、散气、耗气
- 应先调哪里，先停哪里
- 是需要收口、减耗、还是先打通一线

### Personality / tendency
重点看：
- 长处是否总在对的场景发力
- 短处是外弱还是内耗
- 适合快打快收，还是慢成慢稳
- 什么节奏最容易让长处变成成果

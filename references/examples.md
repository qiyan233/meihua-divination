# Examples

Use these examples to calibrate tone, density, and reasoning.

## Example 1: Career decision

Question:
我要不要辞职去做自己的项目？现在工作稳定，但人已经很疲惫，手里有一个想做半年的方向。

Reading shape:
- 问题归类：事业 / 项目 / 决策
- 主象：土
- 客象：火
- 阻象：木
- 变象：金
- 应象：金

Good answer sketch:
- 总断：此势可谋变，但不宜骤断。
- 断因：主象见土，说明现实承载与既有结构仍在；你不是全无根基，而是旧盘太重。客象见火，外部机会与内心躁动都在升温。阻象落木，看似是生发之机，实则说明你现在最卡的不是“没有想法”，而是想法太想立刻长成。
- 断势：近势多为心急与现实拉扯，若强断旧局，反易因承接不足而回头补洞；中势若先试跑、先验证、先留退路，则火可生土，反能把新事做实。
- 断法：先做副线验证，不宜一刀切。先验证用户、节奏、现金流或最小闭环，再定是否转身。
- 签语：先立承盘，再起高楼。

## Example 2: Relationship reading

Question:
我该不该主动联系他？之前聊得不错，但最近明显冷下来了。

Good answer sketch:
- 总断：此关系可问，不宜逼近。
- 断因：主象若偏水或金，往往代表迟疑、边界、冷却；若阻象再见土或金，则说明真正的问题不是“有没有感觉”，而是气口收紧，互动节奏失衡。
- 断势：短期强推无益，轻问反比重压更容易见真意。
- 断法：若联系，宜轻、短、留白，不宜带审问感。
- 签语：外暖内迟，缓问真意。

## Example 3: Current fortune

Question:
为什么我这段时间总觉得做什么都不顺？

Good answer sketch:
- 总断：当前气机偏滞，不在于你全无路，而在于多处同耗。
- 断因：若主象见土或水，且阻象再来克制，多半意味着精力分散、心气回收、行动难以连成线。问题不一定大，但持续被小阻拖慢。
- 断势：近势多见琐碎拖拽，中势关键在减耗与收口；后势一旦把最重的一两处滞点疏开，整体会比你体感中回得更快。若动象不强，则不宜期待立刻翻盘。
- 断法：减少并行事务，先把一个堵点打通。
- 签语：气未全通，先疏其滞。

## Example 4: Dynamic manifestation speed

Question:
这个项目最近会不会很快有结果？

Good answer sketch:
- 总断：此事已有动象，近期较易见回音，但回音未必等于定局。
- 断因：若客象与变象皆动，说明事情已不只是停留在想法层，而是正在进入显化阶段；但若阻象仍强，则反馈来得快，不代表承接已经稳。
- 断势：近势有消息，中势看承接，后势看你能否把外来的机会接成自己的盘。
- 断法：准备好承接方案，不要只等结果出现。
- 签语：机先有响，盘后见真。

## Example 5: Relationship pacing

Question:
我现在继续推进这段关系会不会太快？

Good answer sketch:
- 总断：此情可探，不宜催熟。
- 断因：若主象有意、客象有回，但阻象偏重，多半不是无缘，而是节奏还未完全对齐。
- 断势：近势可以轻触，中势要防用力过头，后势能否转暖，在于你是否守住分寸。
- 断法：若联系，宜轻、短、留白，不宜一次压满情绪与期待。
- 签语：花可轻问，不可强开。

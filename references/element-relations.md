# Element Relations

Use this file to keep the logic of 生、克、泄、耗 consistent.

## 生 (generating)

- 木生火
- 火生土
- 土生金
- 金生水
- 水生木

Interpretation:
- 生 means support, nourishment, continuation, or enabling conditions.
- If 应象生主象, advice aligns well with the user's underlying need.
- If 客象生主象, environment tends to support the matter.

## 克 (controlling)

- 木克土
- 土克水
- 水克火
- 火克金
- 金克木

Interpretation:
- 克 means pressure, restraint, friction, correction, or conflict.
- If 客象克主象, outside forces suppress the matter.
- If 阻象克主象, the bottleneck directly hinders core momentum.
- If 应象克阻象, the recommended response can cut through the bottleneck.

## 泄 / 耗 (draining)

Use these informally in prose when the matter is not exactly controlled but loses force.

Examples:
- 木生火 too much -> enthusiasm burns away stable planning
- 火生土 too much -> action becomes burden and cleanup
- 金生水 too much -> over-analysis weakens decisiveness

Use "泄" or "耗" when the pattern suggests loss of energy rather than direct suppression.

## Fast judgment rules

### 主象 vs 客象
- 生: the environment helps
- 克: the environment constrains
- 同: the environment amplifies the same tendency

### 主象 vs 阻象
- 阻象克主象: obstacle is direct and concrete
- 阻象同主象: self-created bottleneck or internal excess
- 阻象生主象: obstacle looks painful but may contain useful pressure or training

### 主象 vs 变象
- 变象生主象: trend stabilizes the current path
- 变象克主象: trend forces a correction or break
- 变象同主象: tendency intensifies; good or bad depends on balance

### 阻象 vs 应象
- 应象克阻象: direct intervention works
- 应象生阻象: the proposed response may worsen the problem
- 应象同阻象: user may be repeating the same pattern

## Preferred wording

Prefer these phrases in final answers:
- 得生
- 受克
- 同气相引
- 泄气过重
- 反助其阻
- 以应制阻

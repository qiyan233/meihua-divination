# Scenario Patterns

在抽象象数规则之外，优先用这份文件为常见问题提供更像“真人断事”的场景化落点。

## 使用方法

1. 先按 `question-types.md` 判断大类。
2. 再看问题是否落入下列高频场景。
3. 若命中场景，用这里的“重点判断 / 易错点 / 建议落点”去约束输出。
4. 不要让场景模板压过象数结构；场景规则是校准器，不是替代推演。

## 事业 / 项目类

### 1. 辞职 / 转身 / 单干

常见问法：
- 我要不要辞职单干
- 该不该离开现在的工作做自己的项目
- 现在换赛道合适吗

重点判断：
- 主象是否仍有根基，还是已经被旧局耗空
- 客象是机会真到，还是只是躁意先起
- 阻象是现实承接、现金流、执行盘，还是纯心理迟疑
- 变象是越往后越稳，还是越往后越散

易错点：
- 不能把“想走”直接断成“该走”
- 不能把“有火气”当成“时机成熟”
- 不能只谈理想，不谈承接

建议落点：
- 若主象未空、客象有扶、阻象不重，可说“先副线验证，再择机转身”
- 若阻象强、变象又泄，多半要说“先补盘，不宜骤断”

### 2. 合作 / 合伙 / 接项目

常见问法：
- 这个合作要不要接
- 现在和对方一起做是否可行
- 这个项目有没有坑

重点判断：
- 客象是扶主还是压主
- 阻象是否代表边界不清、权责不清、交付不稳
- 应象是否指向“先立规则、先试合作、先小后大”

易错点：
- 不要只看机会大小，要看接不接得住
- 不要把“关系不错”误写成“合作一定顺”

建议落点：
- 多用：先立边界、先做小单、先验配合、再谈放大

### 3. 项目卡住 / 迟迟不出结果

常见问法：
- 为什么项目一直推不动
- 最近做产品为什么没有起色
- 这个方向是不是不对

重点判断：
- 阻象是外压、节奏、目标过散，还是承接盘不足
- 变象若仍可转，说明不是死局，是路径错或耗损过大

建议落点：
- 不急着说“方向错了”
- 更常见的写法是：“方向未必错，错在承接、取舍与节奏”

## 关系类

### 4. 该不该主动联系

常见问法：
- 我要不要主动找他
- 现在联系对方会不会打扰
- 这时候开口合不合适

重点判断：
- 客象是否还有回音
- 阻象是对方有结、节奏未齐，还是你这边情绪过重
- 动象若快，说明一触即有反应，但反应不等于结果好

易错点：
- 不要一上来就断“对方没意思”
- 不要把关系冷下来直接当结束

建议落点：
- 常用结论：可问，不宜逼近；可轻触，不宜压满

### 5. 暧昧 / 推进关系

常见问法：
- 这段关系还能不能往前走
- 我们有没有机会在一起
- 现在推进会不会太快

重点判断：
- 主客是否同频
- 阻象是否显示节奏错位、旧结未清、边界未明
- 变象是转暖、转冷，还是继续拉扯

建议落点：
- 若主客有意、阻象不轻：写成“有意未齐，不宜催熟”
- 若后势有转机：写成“先松结，再谈亲近”

### 6. 复合 / 旧人回头

常见问法：
- 我和前任还有没有可能
- 旧关系还有没有续头
- 他会不会回来

重点判断：
- 阻象是否明显指向旧结未解
- 变象是否只是旧情翻动，还是确有新势可生
- 应象更偏“复盘、清边界、慢试”，还是“宜断不宜续”

易错点：
- 不要把“情未断”写成“必能复合”
- 复合类尤其要防假希望

建议落点：
- 可用：旧气未散，不等于新局已成
- 若要给建议，多写“先辨旧结是否真解”

## 运势 / 阶段气运类

### 7. 最近总是不顺

常见问法：
- 为什么最近做什么都不顺
- 最近是不是运气很差
- 这段时间怎么总卡住

重点判断：
- 阻象是否多为耗气、漏气、并行过多
- 主象是否处于收口、迟滞、回气状态
- 变象是否说明熬过当前收缩后会回升

建议落点：
- 多用：不是大凶，是多处同耗
- 先减耗、先收口、先打通一线

### 8. 这个月适不适合做大动作

常见问法：
- 最近适合换工作吗
- 这个月适合谈合作吗
- 现在适不适合推进大计划

重点判断：
- 动静是否偏快
- 阻象是否强压
- 应象是试跑、蓄势，还是切割旧耗

建议落点：
- 若整体静多于动：说“宜小动，不宜大开”
- 若动象明显但阻重：说“有动机，不宜满推”

## 人设 / 性格 / 天赋类

### 9. 我适合什么路子

常见问法：
- 我适合创业还是稳定发展
- 我适合做内容还是做产品
- 我这类人适合什么节奏

重点判断：
- 主象定气质底色
- 阻象看自耗方式
- 应象看真正适合的工作法，而不只是抽象优缺点

建议落点：
- 不要只写“你很有天赋”这类空话
- 更要写“你适合怎么做，什么情况下容易失衡”

### 10. 自耗 / 节奏失衡型问题

常见问法：
- 为什么我总是想很多却推进很慢
- 我是不是容易自我消耗
- 为什么我一开始很猛，后面就散了

重点判断：
- 阻象是否是典型内耗结构
- 变象是否指向先破后立
- 应象是否在提示“控损耗、减并行、先稳节奏”

建议落点：
- 常用：不是没力量，而是力被自耗吃掉
- 常用：先守节奏，长处才会变成果
